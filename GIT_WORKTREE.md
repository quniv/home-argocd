# Git Worktrees

Managed by `/feat` (open) and `/done` (close).

## Tips
- Each worktree is an independent checkout — open multiple terminals simultaneously.
- Run `/done` inside a worktree directory to close it cleanly.
- `git worktree list` — see all active worktrees.
- `git worktree prune` — remove stale entries after manual deletions.
- Never run `git checkout` inside a worktree — it is locked to its branch.
